## 🧭 How to navigate

- **Browsing a course** → click into any `courses/NN-course-slug/` folder. The course README gives you the summary, key concepts, and an index of assignments and notes.
- **Searching the repo** → use GitHub's search bar (press `/` on any page). It searches across all files.
- **Finding a specific topic** → search the repo or scan the "Key concepts" section in each course README.
